# DeviceDeck

DeviceDeck is a modern tech-service platform built to simplify PC builds, mobile device servicing, and enterprise IT infrastructure solutions through a clean customer-focused experience and a real-time admin management system.

The platform allows users to:

* Book mobile repair and servicing appointments
* Build and customize PCs based on their requirements
* Submit enterprise IT infrastructure inquiries
* Manage profiles, orders, and saved addresses
* Access a smooth modern UI with responsive design

## Core Features

* Firebase Authentication
* Real-time Firestore Database Integration
* Admin Dashboard for Managing Orders & Inquiries
* PC Build Ordering System
* Mobile Service Booking System
* Enterprise IT Inquiry


THE FUNCTIONALITIES OF EACH FILE :

* `index.html` — Main landing page of DeviceDeck containing homepage UI, navigation, services overview, profile drawer system, and customer entry points. 

* `firebase.js` — Central Firebase configuration file handling Authentication, Firestore database connection, Google login provider, and Firebase exports across the project. 

* `login.html` — User login page with Firebase email/password authentication, Google sign-in support, premium split-screen UI, and session handling. 

* `signup.html` — User registration page allowing account creation with Firebase Authentication along with modern onboarding interface and feature highlights. 

* `mobile-services.html` — Mobile repair and service booking page where customers submit repair requests, device details, service type, and booking information connected to Firestore. 

* `pc-builds.html` — Advanced PC build and customization page featuring prebuilt systems, custom PC configurator, component selection, pricing workflow, and order submission system. 

* `it-infrastructure.html` — Enterprise IT solutions page for businesses and institutions to submit infrastructure, networking, server, and organizational IT setup inquiries. 

* `myprofile.html` — Customer profile management dashboard displaying personal information, editable account details, profile sections, and account-related controls. 

* `myorders.html` — Customer order tracking dashboard showing service bookings, PC build orders, statistics cards, and future order status management system. 

* `myaddresses.html` — User address management system for storing, editing, and managing saved delivery and pickup addresses with categorized address cards. 

* `dd-admin-x7k2.html` — Private admin dashboard for managing mobile bookings, PC build orders, enterprise IT inquiries, live Firebase records, analytics sections, and operational workflows.
